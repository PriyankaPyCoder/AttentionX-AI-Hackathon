🎬 AI Content Repurposing Engine

🚀 Problem

Long-form videos (lectures, podcasts, interviews) contain valuable insights, but modern audiences prefer short, engaging content. Important moments often get lost in long videos.

---

💡 Solution

This project converts long-form videos into short, viral clips automatically using AI.

It:

- 🎙 Transcribes video content
- 🔊 Detects important moments
- ✂️ Extracts key clips
- 📱 Converts into short-form (Reels/TikTok style)
- 💬 Adds captions and generates catchy hooks
